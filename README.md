# 🏏 IPL Dashboard (Power BI)

## 📌 Overview

This project presents an interactive Power BI dashboard analyzing IPL (Indian Premier League) data. It provides insights into team performance, player statistics, and match trends.

## 🚀 Key Features

* 📊 Team-wise performance analysis
* 🏆 Top players based on runs and wickets
* 📅 Match trends across seasons
* 📈 Interactive filters and visuals

## 🛠️ Tools & Technologies

* Power BI
* Data Visualization
* Data Cleaning
